OVERVIEW AND SUCCESS

概述與成就

延續 App 設計風格改版，將首頁 UI 做設計一致性的調整並模組化。 App 首頁 UI 重新改版的過程中，不僅是要考慮 UI 的設計，還需與視覺、工程師不斷地溝通、協作與 design review，憑藉著團隊合作才能將更好更完整的 App UI 上線，看到成果當下的感動會覺得之前的努力是值得的。

IDEATION AND DESIGN

構思與設計

使用卡牌的方式區分不同類型的功能與服務，利用主視覺中出現的漸層與大面積的色塊搭配 UI 並導入圓角，呈現更鮮明、活潑與平易近人的生活感，重點訊息利用字體字級的強烈對比，素材圖盡量選用主體明確且背景乾淨，來增加視覺重心並更精準的傳達訊息給使用者。
